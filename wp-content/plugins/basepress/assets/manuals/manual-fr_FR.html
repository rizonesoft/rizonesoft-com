Manuel

Veuillez suivre les instructions ci-dessous pour vous familiariser avec BasePress et mettre en place votre Base de connaissances.

Pour obtenir de l'aide supplémentaire, allez sur notre site web sous Knowledge Base > BasePress.

Configurer la page d'entrée de la Base de connaissances

Avant de commencer à utiliser la Base de connaissances, vous devez créer une nouvelle page standard. Donnez-lui un titre comme "Base de connaissances" (ou tout autre nom sous lequel vous souhaitez appeler votre Base de connaissances), insérez le shortcode [basepress] dedans et publiez la page.

Cette page listera les différents produits avec leurs images et leur description. Vous pouvez ajouter tout le contenu supplémentaire dont vous avez besoin pour alimenter vos Bases de connaissances. Il vous suffit de placer le code court où vous voulez pour que la liste de vos produits apparaisse dans la page.

Si vous avez l'intention de créer une Base de connaissances unique, il vous suffit d'insérer le code court, car cette page ne sera pas affichée. Après avoir créé la page d'entrée, vous devez l'ajouter au menu de votre choix pour que vos visiteurs puissent accéder à votre Base de connaissances. Dans le Tableau de bord, allez à Apparence > Menus et ajoutez la page à votre menu.

Une fois que les liens vers la pages page et le menu sont prêts, vous pouvez effectuer le paramétrage de la Base de connaissances. Allez BasePlace > Paramètres et dans les paramètres généraux, sélectionnez la page que vous venez de créer dans le menu de sélection intitulé "Page de BasePress".

Choisissez une Base de connaissances unique ou multi-produits

BasePress a été conçu pour fonctionner en tant que base de connaissances standard ou multi-produits.

Si vous n'avez qu'un seul produit ou service et que vous n’avez besoin que d’une seule base de connaissances, allez à BasePress > Paramètres et sous les options générales, sélectionnez "Mode produit unique".

Utilisé en mode "Produit unique", BasePress ignore la page d'entrée ainsi les utilisateurs accèdent directement à la Base de connaissances sans que s'affiche la sélection du produit.

Si vous disposez de plus d'un produits ou de services et que vous souhaitez créer une base de connaissances différente pour chacun d'eux, laissez l'option "Mode de produit unique" non sélectionnée et l'utilisateur pourra sélectionner le produit pour lequel il veut l'information.

Choisissez le thème que vous souhaitez utiliser

BasePress est livré avec des thèmes prédéfinis. Vous pouvez choisir celui que vous voulez utiliser à partir de BasePress > Paramètres > Aspect et vous avez en plus accès à quelques options d'aspect général.

Si vous ne l'avez pas encore fait, enregistrez vos paramètres.

Commencez à créer votre contenu

À ce stade, vous pouvez commencer à créer votre contenu et à construire votre ou vos bases de connaissances.

Vous devriez commencer par créer votre premier produit à partir de Base de connaissances > Produits. Si vous avez sélectionné "Mode produit unique" et souhaitez disposer d'une Base de connaissances unique, vous devez tout de même créer un produit et vous devez simplement lui attribuer un nom. Vous pouvez passer du mode mono-produit à multi-produits à tout moment.

Une fois votre premier produit prêt, accédez à Base de connaissances > Sections. Sélectionnez le produit que vous avez créé dans le menu de sélection en haut de l'écran et créez votre / vos section(s).

Maintenant que vous avez créé votre premier produit et votre première section, vous pouvez vous rédiger les articles de la Base de connaissances et créer votre contenu. Avant de publier vos articles, n'oubliez pas de sélectionner le produit et la section à laquelle appartient l'article.

Vous pouvez maintenant voir les pages de votre site Web et cliquer sur le lien "Base de connaissances" dans votre menu. Votre contenu devrait apparaître.

Rappelez-vous qu'il vous faut au moins un produit, une section et un article pour que le contenu s'affiche sinon vous verrez simplement une page vide.
